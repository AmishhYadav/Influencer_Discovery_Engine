================================================================================
                         API ENDPOINTS REQUIRED
================================================================================

GET /api/search
  Parameters: keywords, platforms, min_followers, max_followers, 
              min_engagement, max_engagement, category
  Returns: { creators: [], total: number, page: number, per_page: number }

GET /api/creators/:id
  Returns: CreatorDetail object

GET /api/creators
  Parameters: page, per_page
  Returns: { creators: [], total: number, page: number, per_page: number }

GET /api/filter-options
  Returns: { platforms: [], categories: [] }
