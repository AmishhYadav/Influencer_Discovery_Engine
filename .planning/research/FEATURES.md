# FEATURES: Influencer Discovery Engine

## Table Stakes (Must Have for MVP)
- **YouTube Ingestion Engine**: Ability to input a channel/video and pull transcripts, tags, and audience metrics.
- **Semantic Alignment Scoring**: NLP-based scoring to determine how closely a creator's content matches target topics (e.g., plant-based health, sustainability) without explicit activist keywords.
- **Creator Dashboard**: UI to view ranked candidates, filter by alignment score or audience size, and view their metrics.
- **Outreach Briefing Generator**: AI module that takes a creator's profile and outputs a structured, bulleted cheat sheet for human coordinators.

## Differentiators (Competitive Advantage)
- **"Soft Activism" Detection**: The specific algorithm/prompt chain that successfully identifies values-aligned professionals while actively filtering out polarizing/explicit activists. This is the core IP.
- **Contextual Evidence**: Surfacing the exact quotes or video timestamps that justify the alignment score, building trust with the outreach coordinator.

## Anti-Features (Do NOT Build for MVP)
- **Automated Email Outreach**: Do not build automated messaging. The system empowers human coordinators; it does not replace them.
- **Cross-Platform Ingestion**: Do not build scrapers for Instagram, TikTok, or blogs yet. Prove the value on YouTube transcripts first.
- **Complex CRM**: Do not build a full relationship management tool. The MVP should focus on *discovery* and *briefing*, integrating with existing CRMs later.
