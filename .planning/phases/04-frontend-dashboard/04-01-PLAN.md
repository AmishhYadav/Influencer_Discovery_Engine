# Phase 4 Plan 01: Frontend Foundation

## Objective
- **What:** Scaffold the React SPA, install styling tooling (Tailwind + Shadcn), and set up API fetching structure (React Query).
- **Why:** Establishes the boilerplate needed before building the UI components.
- **Output:** A new `frontend` director with Vite, Tailwind, Shadcn, and Tanstack Query installed.

## Tasks

<task type="command">
  <name>4-01-01: Scaffold Vite App</name>
  <action>Run `npm create vite@latest frontend -- --template react-ts`. CD into `frontend`, run `npm install`. Install `react-router-dom` and `@tanstack/react-query`.</action>
  <verify>Run `npm run dev` and ensure `localhost:5173` loads.</verify>
  <done>Base Vite TypeScript app is created.</done>
</task>

<task type="command">
  <name>4-01-02: Tailwind & Shadcn Config</name>
  <action>In `frontend`, install Tailwind: `npm install -D tailwindcss postcss autoprefixer`, then `npx tailwindcss init -p`. Configure `tailwind.config.js` and `src/index.css`. Run `npx shadcn-ui@latest init` (using default settings: New York style, Zinc color, css variables yes). Add some base components: `npx shadcn-ui@latest add button table card sheet badge input skeleton progress`.</action>
  <verify>Verify `components/ui` folder is populated.</verify>
  <done>Styling system is ready.</done>
</task>

<task type="code">
  <name>4-01-03: App Providers & Layout</name>
  <files>frontend/src/main.tsx, frontend/src/App.tsx, frontend/src/components/Layout.tsx</files>
  <action>Wrap main app with `QueryClientProvider` and `BrowserRouter`. Create a basic `Layout.tsx` with a top navigation bar ("Influencer Discovery Engine"). Setup a `.env` file with `VITE_API_URL=http://localhost:8000`.</action>
  <verify>Syntax check.</verify>
  <done>Providers injected and app is ready for views.</done>
</task>
