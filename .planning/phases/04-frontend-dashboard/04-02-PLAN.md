## Objective
- **What:** Build the UI components (Data Table, Detail Sheet) and wire them to the FastAPI backend using React Query.
- **Why:** This fulfills the core visual requirements of the project.
- **Output:** API client functions, React views, and UI workflows in the `frontend` directory.
