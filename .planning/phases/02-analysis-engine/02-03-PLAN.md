## Objective
- **What:** Write a CLI script `analyze.py` that processes a channel's videos, scores them, and updates the database.
- **Why:** This wraps the chunking, embedding, and LLM scoring modules into an executable workflow.
- **Output:** `analyze.py` script and integration tests.
