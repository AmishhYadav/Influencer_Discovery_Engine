# Phase 1 Plan 02: Storage and CLI Integration

## Objective
- **What:** Configure PostgreSQL JSONB storage and create the main CLI entrypoint.
- **Why:** We need a way to persist the ingested data and a tool to trigger the process end-to-end.
- **Output:** Database models, migration/setup scripts, and `ingest.py` CLI script.
